## рҹҸ—пёҸ Architecture

**Core components:**
 
- **Service Discovery** вҖ” Eureka Service Registry handles dynamic service lookup, removing the need for hardcoded service URLs.
- **API Gateway** вҖ” Spring Cloud Gateway centralizes routing, request filtering, and acts as the single entry point for the frontend.
- **Domain Services** вҖ” Independently deployable services for Order & Cart, Product Catalog, and User Authentication (JWT + RBAC).
- **Orchestration** вҖ” `docker-compose.yml` spins up the full backend stack (services + database) for local development with a single command.
